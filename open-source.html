Open Source

What is Open Source?

Open source is simply licensing your work so it can be used how you want it.
Open source software is code that is designed to be publicly accessible—anyone can see, modify, and distribute the code as they see fit.
It is developed in a decentralized and collaborative way, relying on peer review and community production.

Open Source vs Free Software

The two terms are often interchanged however, they are not the same.
Open source does not mean Free software likewise Free software does not mean open source.
Free software does not grant access to the source code. It is also illegal to modify due to license On the other hand, One can charge for his/her open source software.

Why Use and promote open source?

There are numerous reasons why we use and promote open source. They include: Peer review
Ability to be reused and discoverability for your work.
Ability to be intergrated into other system
Distributed Innovation
Colllaboration
Documentation
Minimizing risks associated with closed software such as bugs not easily identified



How to know if a software is open source

Not every code viewable online is open source.
Open source sofware is a software that includes a license. Without the presence of a license, the software is not open source. This also applies to data, photos, slides, etc



Publishing and sharing open source code

In publishing and sharing open source code, we make use of version control systems.
